The developer, the studio, and how I work.

// who

I'm viz. I build software under the name AbstractOcclusion — a solo studio for graphics, systems and the tools that sit between them. I've been writing code for 30+ years, the last 20 professionally, and I work in both English and French.

My comfort zone is low-level: C and C++ for engines and performance-critical code, C# for tooling and gameplay, Python for prototyping, pipelines and the scripts that glue everything together. When a project needs something else, I pick it up.

I like problems where the constraints are real — frame budgets, memory, latency — and where the result is something you can see move on screen.

// the name

Ambient occlusion is the soft shadow where surfaces meet — the detail your eye reads without noticing. AbstractOcclusion takes that idea sideways: the hidden structure underneath is what makes the surface believable.

That's how I like to build. The interesting work is usually the part nobody sees.
